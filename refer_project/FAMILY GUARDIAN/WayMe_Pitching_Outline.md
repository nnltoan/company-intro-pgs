## 1. Introduction of WayMe
*Highlighting our understanding of WayMe's vision, core values, and current challenges.*

- **Vision:** To become the #1 outdoor companion app in safety, navigation, and live tracking.
- **Core Value:** Empowering individuals and communities—hikers, runners, and outdoor enthusiasts—to explore the outdoors confidently and safely.
- **Key Offerings:** Advanced geofencing, off-route alerts, real-time location sharing, curated trails database, and progress monitoring.
- **Current Objective:** Developing a robust Live Tracking ecosystem that integrates the WayMe app with external GPS trackers, allowing for phone-free tracking and real-time visualization on web platforms.

## 3. Potential Projects (WayMe x DanaExperts)
*Mapping WayMe's specific tasks to DanaExperts' technological strengths.*

- **Phase 1: GPS Hardware-to-App Integration**
  - Replacing standard GPS trackers with the WayMe App.
  - Developing solutions to pair external GPS trackers with WayMe so they can operate offline or without a phone (leveraging our IoT/Embedded expertise).
- **Phase 2: Live Tracking Service & Platform Connectivity**
  - Backend integration using Node.js to connect an existing Live Tracking Service seamlessly.
  - Developing data pipelines to feed real-time coordinates of WayMe users securely to the web interface.
  - Using **React, TypeScript, and Mapbox GL JS** to visualize tracking data dynamically on the WayMe website.
- **End-to-End Tech Stack Match:** 
  - DanaExperts natively works with WayMe's exact tech stack: **React, Mapbox GL JS, Node.js, Firebase Firestore, MongoDB, AWS Suite (Cognito, EC2), and Docker CI/CD**.
